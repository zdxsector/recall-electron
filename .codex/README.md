# Codex Adapter

This directory contains Codex-specific adapters for the canonical AI framework in `.agents/`.

Rules:

- canonical role definitions live in `.agents/agents/*.md`
- canonical skills live in `.agents/skills/*/SKILL.md`
- Codex files here stay short and execution-oriented
- keep `config.toml` aligned with `AGENTS.md`
- Playwright Electron testing guidance should remain centralized in the canonical skill
